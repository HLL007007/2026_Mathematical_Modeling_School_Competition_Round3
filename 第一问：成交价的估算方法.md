## 第一问：成交价的估算方法

### 1. 已达成协议国家/地区的转播权成交价估算方法分析

#### 1.1 估算对象的选择与基准数据溯源
在本问中，我们选取**韩国**作为研究对象来剖析转播权的定价估算方法。选取韩国的核心原因在于其数据展现出了极强的**“反直觉与博弈性”**：2026年美加墨世界杯的比赛时间在韩国当地属于极度非黄金的凌晨时段，但在这种恶劣的收视条件下，韩国的成交价却从上一届的 0.55 亿美元暴涨至 1.25 亿美元。解析这一异常暴涨背后的估算逻辑，有助于为后续建立普适性定价模型提供关键参数。

为了确保估算基准的可靠性，我们对韩国历史转播费进行了溯源。根据韩联社（2018年）与《首尔体育》（2019年）的公开报道，韩国三大公共电视台（KBS、MBC、SBS）曾组成购买联合体，以打包形式支付了 2018 年与 2022 年两届世界杯的版权费，折算单届成本均稳定在 **0.55 亿美元**。由于 2018 与 2022 两届赛事的容量（64场）与韩国收视时差（UTC+3时区，评分极近）基本一致，该平稳数据在数学上验证了：**当核心宏观条件不变时，版权价格具有强历史延续性。**

因此，我们确立 2022 年的 0.55 亿美元为基准锚点（Anchor），采用**“历史基准修正法（Historical Baseline Method）”**来反推 2026 年成交价的估算逻辑。

#### 1.2 估价模型的建立：历史-增量修正模型
版权购买方在评估下一周期价格时，通常以历史成交价为基准，并叠加各项核心变量的预期增量。我们建立如下的乘法修正方程：

$$ P_{2026} = P_{2022} \cdot \left(\frac{V_{2026}}{V_{2022}}\right)^\alpha \cdot \left(\frac{Z_{2026}}{Z_{2022}}\right)^\beta \cdot M_{market} \cdot C_{comp} $$

其中各项参数的经济学与数学含义如下：
*   $P_{2026}, P_{2022}$：当届与基准届的转播权成交价。
*   $V$ (Volume)：赛事容量。比赛场次由 64 场增至 104 场。$\alpha$ 为**扩军价值弹性系数**，衡量新增场次的边际效用。
*   $Z$ (Timezone)：时差评分。2022年卡塔尔对韩国评分为 0.9（包含部分黄金时间），2026年降为 0.5（凌晨）。$\beta$ 为**时差折损敏感系数**。
*   $M_{market}$：宏观市场增值乘数。由绝对球迷基数增长率与人均GDP增长率的乘积构成。
*   $C_{comp}$：买方市场竞争溢价指数。衡量区域内转播商的博弈状态（买方卡特尔 vs. 自由竞价）。

#### 1.3 核心数据的代入与参数推导
根据赛会信息与宏观经济数据表，我们将韩国的相关数据代入模型进行计算：
1.  **赛事扩容率**：$V_{ratio} = 104 / 64 = 1.625$
2.  **时差折损率**：$Z_{ratio} = 0.5 / 0.9 \approx 0.555$
3.  **市场增量乘数**：根据测算，2026年韩国绝对球迷人数对比2022年增长约 $5.6\%$，人均GDP增长约 $7.1\%$，得出综合市场乘数 $M_{market} = 1.056 \times 1.071 \approx 1.131$。

将上述结果及 $P_{2026} = 1.25, P_{2022} = 0.55$ 代入核心方程并化简得：

$$ 1.25 = 0.55 \times (1.625)^\alpha \times (0.555)^\beta \times 1.131 \times C_{comp} $$
$$ (1.625)^\alpha \times (0.555)^\beta \times C_{comp} \approx 2.009 $$

在此，我们通过控制变量法，绘制出要使该估价等式成立，扩军弹性 $\alpha$ 与时差敏感度 $\beta$ 的可行解组合：

#### 1.4 估算结果的商业博弈解释（闭环分析）
从数学方程的求解结果可以看出，要使目标乘数达到 2.009（即价格暴涨 2.27 倍），在时差大幅度恶化（底数为0.555）的约束下，**必须满足时差敏感系数 $\beta \to 0$ 且竞争溢价 $C_{comp} > 1$**。这一数学结论完美契合了2026年韩国区域转播权谈判的现实商业逻辑：

1.  **竞争格局的突变（破除买方垄断，推高 $C_{comp}$）**：
    2018与2022年，韩国三大台组成“买方卡特尔”，消除了内部竞争（$C_{comp} \le 1$），成功将价格压制在 0.55 亿。而2026年，JTBC 跨界独立竞标，直接打破了垄断平衡。FIFA 利用“鹬蚌相争”的博弈格局，成功获取了高额的竞争溢价（Premium）。
2.  **商业模式的降维打击（时差敏感度 $\beta$ 趋近于0）**：
    传统的免费国家台依赖“黄金时段直播”来赚取广告费，极度惧怕时差劣势。但 JTBC 是一家依赖“付费会员与视频点播(VOD)”的综合频道/流媒体平台。对于付费订阅模式，凌晨直播劣势可通过“次日高清回放/集锦”被大幅抹平。因此，模型中的 $\beta$ 值极小，时差惩罚失效。
3.  **扩军的边际效益最大化（高 $\alpha$ 值）**：
    增加的 40 场比赛对于传统电视台是难以排播的负担，但对于 JTBC 这类流媒体而言，是极度丰厚的内容库（Content Library），能显著拉动长尾订阅。

**本问小结：** 
通过对韩国区 1.25 亿美元成交价的剖析，我们得出，转播权估价不能仅依据简单的线性外推（如单纯计算 GDP 和人口）。**实际成交价是由基础宏观数据托底，由赛事容量增幅提供上行动力，并受到该地区“媒体竞争格局”与“转播商商业模式”深刻调节的博弈结果。**这一估算逻辑为我们在下一问建立全局定价模型奠定了理论基础。

```python
import numpy as np
import pandas as pd
import matplotlib.pyplot as plt

# ================= 字体设置 (防止中文乱码) =================
# Windows 用户默认使用 SimHei
plt.rcParams['font.sans-serif'] = ['SimHei']  
# Mac 用户如果报错，请注释掉上一行，取消下方这行的注释：
# plt.rcParams['font.sans-serif'] = ['Arial Unicode MS'] 

plt.rcParams['axes.unicode_minus'] = False # 正常显示负号

# ================= 数据输入 =================
# 1. 韩国 2018-2026 核心数据提取
data_kor = {
    'Year': ['2018(俄罗斯)', '2022(卡塔尔)', '2026(美加墨)'],
    'Price': [0.55, 0.55, 1.25],          # 成交价(亿美元)
    'Matches': [64, 64, 104],             # 比赛容量
    'Timezone_Score': [0.8, 0.9, 0.5]     # 时差评分
}
df = pd.DataFrame(data_kor)


# ================= 图1：韩国历届转播费与比赛容量趋势图 =================
fig, ax1 = plt.subplots(figsize=(9, 5.5))

# 绘制价格柱状图 (主Y轴)
bars = ax1.bar(df['Year'], df['Price'], color=['#7fb3d5', '#5499c7', '#2980b9'], width=0.4, label='转播权成交价 (亿美元)')
ax1.set_ylabel('成交价 (亿美元)', fontsize=12, fontweight='bold')
ax1.set_ylim(0, 1.6)

# 为柱子添加具体数值标签
for bar in bars:
    yval = bar.get_height()
    ax1.text(bar.get_x() + bar.get_width()/2, yval + 0.02, f"{yval}亿", ha='center', va='bottom', fontsize=12, fontweight='bold')

# 绘制场次折线图 (副Y轴)
ax2 = ax1.twinx()
line = ax2.plot(df['Year'], df['Matches'], color='#e74c3c', marker='o', linewidth=2.5, markersize=8, label='比赛场次容量 (场)')
ax2.set_ylabel('比赛场次容量 (场)', fontsize=12, fontweight='bold', color='#e74c3c')
ax2.set_ylim(50, 120)
ax2.tick_params(axis='y', labelcolor='#e74c3c')

plt.title('图1：韩国历届世界杯转播费趋势与容量平稳性检验', fontsize=14, fontweight='bold')

# 合并图例并调整位置
lines_1, labels_1 = ax1.get_legend_handles_labels()
lines_2, labels_2 = ax2.get_legend_handles_labels()
ax1.legend(lines_1 + lines_2, labels_1 + labels_2, loc='upper left')

plt.grid(axis='y', linestyle='--', alpha=0.6)
plt.tight_layout()
plt.show() # 显示图1


# ================= 图2：估价模型参数反推与博弈分析 =================
# 基于修正模型: P_2026 = P_2022 * (V_ratio)^alpha * (Z_ratio)^beta * M_market * C_comp
P_2022 = 0.55
P_2026 = 1.25
V_ratio = 104 / 64          # 容量扩军率 = 1.625
Z_ratio = 0.5 / 0.9         # 时差恶化率 = 0.555
M_market = 1.056 * 1.071    # 综合宏观增量(球迷增量 * GDP增量) = 1.131

# 目标常数: (1.625)^alpha * (0.555)^beta * C_comp = 2.009
target_val = P_2026 / (P_2022 * M_market) 

alphas = np.linspace(0.5, 3.0, 100) # 假设场次价值弹性 alpha 的合理横轴区间

# 求解函数：根据不同的竞争格局(C_comp)，求解对应的 Beta 曲线
def calculate_beta(alpha, C_comp):
    # 数学推导: beta = [ln(target) - ln(C_comp) - alpha * ln(V_ratio)] / ln(Z_ratio)
    return (np.log(target_val) - np.log(C_comp) - alpha * np.log(V_ratio)) / np.log(Z_ratio)

# 场景A：假设韩国依然是三大台垄断，无竞争溢价 (C_comp = 1.0)
betas_monopoly = calculate_beta(alphas, 1.0)
# 场景B：现实情况，JTBC独立竞标引发价格战，带来约30%的竞争溢价 (C_comp = 1.3)
betas_competitive = calculate_beta(alphas, 1.3)

plt.figure(figsize=(9, 6))
plt.plot(alphas, betas_monopoly, 'b-', linewidth=2.5, alpha=0.6, label='理论场景：维持买方垄断 (C_comp=1.0)')
plt.plot(alphas, betas_competitive, 'r-', linewidth=2.5, label='真实场景：引入自由竞争破除垄断 (C_comp=1.3)')

# 绘制辅助线 (X轴=1, Y轴=0)
plt.axhline(y=0, color='gray', linestyle='--', alpha=0.8)
plt.axvline(x=1, color='gray', linestyle='--', alpha=0.8)

# 重点标注符合 JTBC(流媒体) 商业模式的逻辑解区间
# 流媒体特征：内容越多越好 (Alpha > 1.2)，且不在乎半夜直播(Beta 趋近于0)
plt.fill_between(alphas, -0.5, 0.5, where=(alphas >= 1.2) & (alphas <= 2.5), 
                 color='green', alpha=0.15, 
                 label='JTBC流媒体商业模式的可行解区间\n(高扩军收益，极低时差敏感度)')

plt.xlabel('赛事扩军价值弹性 (α)', fontsize=12, fontweight='bold')
plt.ylabel('时差折损敏感度 (β)', fontsize=12, fontweight='bold')
plt.title('图2：2026年估价模型：场次增量(α)、时差折损(β)与竞争格局的博弈', fontsize=14, fontweight='bold')
plt.ylim(-1.5, 3)
plt.legend(loc='upper right', fontsize=10)
plt.grid(True, linestyle=':', alpha=0.7)
plt.tight_layout()
plt.show() # 显示图2
```
