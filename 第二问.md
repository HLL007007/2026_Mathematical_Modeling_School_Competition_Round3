## 2.FIFA 区域市场转播权普适性定价模型的建立

### **2.1 模型选择与理论基础：基于柯布-道格拉斯的扩展模型**
区域转播权的市场定价受到宏观经济、人口基数、赛事特征等多重因素的非线性交叉影响。传统的多元线性加法模型（$Y = \sum a_i X_i$）无法反映经济学中的“乘数效应”与“一票否决”规律（例如：当某地区的足球受众基数趋近于零时，其人均GDP再高，转播价值也应趋近于零，加法模型无法实现这一点）。

因此，本研究引入微观经济学中经典的**柯布-道格拉斯生产函数（Cobb-Douglas Function）**，将其扩展为适用于版权定价的**多元指数乘法模型**。该模型假设各影响因子对最终价格的贡献是连乘关系，不仅完美契合现实商业逻辑，更能通过对数线性化，精准提取各变量的“价值弹性（Elasticity）”。

### **2.2 核心变量选取与数据清洗（Data Preprocessing）**
基于公开报道数据，我们提取了以下 5 个决定转播价值的核心连续变量：

1.  **绝对球迷基数 ($Fans$)**：代表基础市场盘口大小（区域总人口 × 足球关注度）。
2.  **人均GDP ($GDP\_pc$)**：代表该国受众的单客广告消费力与购买力。
3.  **时差评分 ($Timezone$)**：衡量比赛时间与当地黄金收视时段的重合度（0.5~1.2）。
4.  **赛事容量 ($Matches$)**：即比赛场次，衡量转播商获得的绝对内容资产量。
5.  **买方竞争指数 ($Comp$)**：衡量区域转播商处于买方垄断（0.7）还是激烈竞价（1.2+）状态。

*【数据预处理与特征工程说明】*：
在构建全局面板数据时，我们特别进行了两项严谨的科学过滤，以确保模型的普适性：
第一，**剔除美国结构性异常值**：美国体育版权市场高度依赖高昂的有线电视订阅费与拉美裔高溢价，其定价逻辑完全独立于欧亚大陆。若保留美国极端数据，将严重导致全局回归超平面倾斜，故予以剔除。
第二，**剔除哑变量共线性干扰**：初步检验发现，由于中国拥有极其庞大的绝对球迷基数且往届未参赛，这在算法底层与“本国球队是否参赛($Team$)”这一特征产生了强烈的混淆效应（Confounding Effect）。为保证因果推断的准确性，我们将 $Team$ 变量剥离，专注于绝对市场容量与商业博弈的量化。

### **2.3 核心定价方程的构建与参数求解**
我们建立的基础非线性定价方程为：
$$ Price = e^{\beta_0} \cdot (Fans)^{\beta_1} \cdot (GDP\_pc)^{\beta_2} \cdot (Timezone)^{\beta_3} \cdot (Matches)^{\beta_4} \cdot (Comp)^{\beta_5} $$

为使用普通最小二乘法（OLS）进行参数估计，我们对方程两边同时取自然对数（$\ln$），转化为**多元对数线性回归模型**：
$$ \ln(Price) = \beta_0 + \beta_1 \ln(Fans) + \beta_2 \ln(GDP\_pc) + \beta_3 \ln(Timezone) + \beta_4 \ln(Matches) + \beta_5 \ln(Comp) + \epsilon $$

我们将经过清洗的 8 个国家/地区（英、日、韩、德、越、巴、澳、中国香港）在 2018、2022、2026 三届世界杯的 24 组面板数据代入 Python 的 `statsmodels` 模块进行拟合。

### **2.4 模型拟合精度检验**
拟合结果显示，该多元对数线性模型的解释力表现优异：决定系数 $R^2$ 达到了 **0.7470**。这意味着在现实世界复杂的跨国商业博弈中，我们选取的 5 个核心宏观与行业变量，成功解释了全球区域市场转播权价格方差的近 75%，证明了所选特征的科学性与有效性。

如上方的模型检验散点图所示，各样本国家（包括英、日、韩、德等）的“模型理论预测价”与“实际公开报道成交价”较好地沿 $y=x$ 的完美预测虚线均匀分布，未出现严重的系统性偏离或明显的异方差性。这证明我们建立的模型在剥离了单一市场的极端特殊数据（如美国特有的高昂有线电视订阅溢价结构）后，成功捕捉到了区域定价的底层商业逻辑，具备良好的全球普适性与估价合理性。

结合回归结果，我们确立的 FIFA 全局标准定价公式为：
$$ Price = e^{-4.3909} \cdot (Fans)^{0.8528} \cdot (GDP\_pc)^{0.6489} \cdot (Timezone)^{0.1087} \cdot (Matches)^{0.0937} \cdot (Comp)^{1.9602} $$

### **2.5 定价因子的价值弹性（敏感度）分析**

多元对数模型的一大数学优势在于，其拟合出的参数具有明确的微观经济学意义，即**价值弹性系数（自变量每变动 1%，引起因变量变动的百分比）**。我们对各因子的弹性进行了可视化诊断。

<img src="D:\黄洛霖的文件\竞赛\数模赛\2026_Mathematical_Modeling_School_Competition_Round3\图四因子价值弹性分析.png" style="zoom: 67%;" />

通过对弹性系数的横向对比，我们得出以下深刻的商业定价洞察，这为理解 FIFA 的报价行为提供了理论支撑：

1. **“竞争”是最大的溢价杠杆**：买方竞争格局（$Comp$）的弹性系数遥遥领先，高达 **1.960**。这意味着竞争指数每增加 1%，理论转播费将呈现近两倍的非线性增长。这在数学上确证了：打破买方垄断、引入多平台竞标（或促使多平台联合出资）是推高转播费的最核心动力。
2. **“球迷基数”与“购买力”构成双擎支撑**：绝对球迷基数（$Fans$）的弹性为 **0.853**，人均GDP消费力（$GDP\_pc$）的弹性为 **0.649**。这说明庞大的受众基本盘依然是转播版权价值的第一大基石，而当地较高的经济水平（意味着更高的单客广告变现价值 ARPU）则提供了稳健的购买力支撑。两者共同决定了该区域的基础商业天花板。
3. **扩军增量能部分对冲时差劣势**：收视时差重合度（$Timezone$，**0.109**）与比赛扩容场次（$Matches$，**0.094**）均呈现较小的正向弹性。这表明，虽然糟糕的时差会拉低报价，但 FIFA 通过“强行扩军至104场（增加绝对内容资产量）”的手段，确实能够在模型估值上产生一定的正向拉动，从而试图在总量上稳住甚至推高最终成交价。

### **本问小结：**

我们成功构建了一个普适全球的非线性动态定价模型。该模型不再是依靠主观直觉的“漫天要价”，而是严格基于宏观经济基本面与传媒竞争格局的量化工具。在下一问中，我们将应用该模型，代入中国区的各项参数，对2026年中国内地的转播权进行精准的理论报价测算，并分析现有争议。

```python
import pandas as pd
import numpy as np
import statsmodels.api as sm
import matplotlib.pyplot as plt
import seaborn as sns

plt.rcParams['font.sans-serif'] = ['SimHei']  
plt.rcParams['axes.unicode_minus'] = False 

# 1. 修正后的数据集：
data_refined = [
    # 2018
    ['英国', 2018, 1.07, 29.93, 4.36, 1.2, 64, 1.0],
    ['日本', 2018, 1.36, 31.63, 3.91, 0.8, 64, 1.3],
    ['韩国', 2018, 0.55, 21.16, 3.33, 0.8, 64, 1.2],
    ['德国', 2018, 1.24, 33.99, 4.79, 1.2, 64, 1.2],
    ['越南', 2018, 0.12, 74.49, 0.32, 0.8, 64, 0.7],
    ['巴西', 2018, 2.00, 125.64, 0.92, 1.0, 64, 1.0],
    ['澳大利亚', 2018, 0.15, 6.97, 5.74, 0.7, 64, 1.0],
    ['中国香港', 2018, 0.26, 4.13, 4.80, 0.8, 64, 0.9],
    ['中国', 2018, 1.50, 306.99, 1.00, 0.8, 64, 0.7],
    # 2022
    ['英国', 2022, 0.99, 30.15, 4.63, 1.2, 64, 1.0],
    ['日本', 2022, 1.14, 32.53, 3.41, 0.9, 64, 1.3],
    ['韩国', 2022, 0.55, 20.68, 3.23, 0.9, 64, 1.2],
    ['德国', 2022, 1.11, 35.20, 4.87, 1.2, 64, 1.2],
    ['越南', 2022, 0.15, 78.56, 0.42, 0.9, 64, 0.7],
    ['巴西', 2022, 2.20, 133.49, 0.91, 1.0, 64, 1.0],
    ['澳大利亚', 2022, 0.14, 7.80, 6.50, 0.8, 64, 1.0],
    ['中国香港', 2022, 0.38, 4.07, 4.86, 0.9, 64, 0.9],
    ['中国', 2022, 1.50, 338.88, 1.27, 0.9, 64, 0.7],
    # 2026
    ['英国', 2026, 1.75, 31.28, 5.44, 0.6, 104, 1.0],
    ['日本', 2026, 2.00, 34.16, 3.57, 0.5, 104, 1.3],
    ['韩国', 2026, 1.25, 21.84, 3.46, 0.5, 104, 1.2],
    ['德国', 2026, 1.40, 36.96, 5.60, 0.6, 104, 1.2],
    ['越南', 2026, 0.15, 83.83, 0.50, 0.5, 104, 0.7],
    ['巴西', 2026, 1.10, 141.70, 1.03, 1.2, 104, 1.0],
    ['澳大利亚', 2026, 0.15, 8.91, 6.67, 0.6, 104, 1.0],
    ['中国香港', 2026, 0.25, 4.41, 5.13, 0.5, 104, 0.9]
]

cols = ['Country', 'Year', 'Price', 'Fans', 'GDP_pc', 'Timezone', 'Matches', 'Comp']
df = pd.DataFrame(data_refined, columns=cols)

# 2. 对数化处理
df['ln_Price'] = np.log(df['Price'])
df['ln_Fans'] = np.log(df['Fans'])
df['ln_GDP_pc'] = np.log(df['GDP_pc'])
df['ln_Timezone'] = np.log(df['Timezone'])
df['ln_Matches'] = np.log(df['Matches'])
df['ln_Comp'] = np.log(df['Comp'])

# 3. 构建 OLS 回归
X = df[['ln_Fans', 'ln_GDP_pc', 'ln_Timezone', 'ln_Matches', 'ln_Comp']]
X = sm.add_constant(X) 
Y = df['ln_Price']
model = sm.OLS(Y, X).fit()

# 打印结果
print("模型参数：")
print(f"R-squared: {model.rsquared:.4f}")
print(f"Intercept(beta_0): {model.params['const']:.4f}")
print(f"Fans(beta_1): {model.params['ln_Fans']:.4f}")
print(f"GDP_pc(beta_2): {model.params['ln_GDP_pc']:.4f}")
print(f"Timezone(beta_3): {model.params['ln_Timezone']:.4f}")
print(f"Matches(beta_4): {model.params['ln_Matches']:.4f}")
print(f"Comp(beta_5): {model.params['ln_Comp']:.4f}")

# 4. 生成的预测图
df['Predicted_ln_Price'] = model.predict(X)
df['Predicted_Price'] = np.exp(df['Predicted_ln_Price'])

plt.figure(figsize=(9, 6))
sns.scatterplot(x='Price', y='Predicted_Price', hue='Country', data=df, s=120, palette='tab10', edgecolor='black', alpha=0.9)

max_val = max(df['Price'].max(), df['Predicted_Price'].max())
plt.plot([0, max_val + 0.2], [0, max_val + 0.2], 'r--', lw=2, label='完美预测线 (y=x)')

plt.title('定价模型检验', fontsize=15, fontweight='bold')
plt.xlabel('实际公开报道成交价 (亿美元)', fontsize=12, fontweight='bold')
plt.ylabel('模型理论预测价 (亿美元)', fontsize=12, fontweight='bold')
plt.legend(bbox_to_anchor=(1.05, 1), loc='upper left')
plt.grid(True, linestyle='--', alpha=0.5)
plt.tight_layout()
plt.show()

# 5. 生成的弹性系数图
elasticities = {
    '绝对球迷基数 (Fans)': model.params['ln_Fans'],
    '人均GDP消费力 (GDP_pc)': model.params['ln_GDP_pc'],
    '买方竞争格局 (Comp)': model.params['ln_Comp'],
    '比赛扩容场次 (Matches)': model.params['ln_Matches'],
    '收视时差重合度 (Timezone)': model.params['ln_Timezone']
}

coef_df = pd.DataFrame(list(elasticities.items()), columns=['Factor', 'Elasticity']).sort_values(by='Elasticity', ascending=True)

plt.figure(figsize=(9, 5))
bars = plt.barh(coef_df['Factor'], coef_df['Elasticity'], color='#2ecc71')
plt.axvline(x=0, color='black', linestyle='-', linewidth=1.5)

plt.title('全球定价模型：因子价值弹性分析', fontsize=15, fontweight='bold')
plt.xlabel('价值弹性系数', fontsize=12, fontweight='bold')

for bar in bars:
    width = bar.get_width()
    plt.text(width + 0.05, bar.get_y() + bar.get_height()/2, f"{width:.3f}", va='center', ha='left', fontweight='bold')

plt.grid(axis='x', linestyle=':', alpha=0.6)
plt.tight_layout()
plt.show()
```
